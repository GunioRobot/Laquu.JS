Laquu.JS バブルポップアップ

シンプルなバブルポップアップ

Code

$(function(){
    laquu("#bubble1").bubblepopup();
});

Bubblepopup trigger.

これがバブルポップアップ

邪魔なコンテンツを整理することが出来ます。

オプション付きバブルポップアップ

Code

$(function(){
    laquu("#bubble2").bubblepopup({
        easing: "easeInOutBack",
        distance: 50,
        distanceStep: 10,
        hideDelay: 600,
        popupClass: ".popup",
        triggerClass: ".trigger",
        onShow: function(popup, trigger) {
            popup.animate({ fontSize: "2.4em" });
        },
        onHide: function(popup, trigger) {
            popup.animate({ fontSize: "100%" });
        }
    });
});

Bubblepopup trigger.

これがバブルポップアップ

邪魔なコンテンツを整理することが出来ます。

利用可能オプション

easing
ポップアップのアニメーションイージングを指定します
distance
ポップアップとトリガーの距離を指定します
step
ポップアップの移動値を指定します
hideDelay
マウスがトリガーから離れてポップアップが消えるまでの遅延時間を指定します
popupClass
ポップアップとして動作させる要素のクラスを指定します
triggerClass
トリガーとして動作させる要素のクラスを指定します
onShow
ポップアップが表示された後に実行されるコールバック関数です
onHide
ポップアップが消えた後に実行されるコールバック関数です
