オプション付きオパシティオーバー

Code

jQuery(function(){
    laquu("a").opacityOver({
        opacity: .4,
        speed: 200,
        onComplete: function(){
            $(this).css("border", "1px solid #F00");
        },
        onHover: funciton(){
            $(this).text("オン！");
        },
        onOut: function(){
            $(this).text("リンク");
            $(this).css("border", "none");
        }
    });
});

opacity
透明度を指定します。
speed
アニメーション速度を指定します。
最低値は50で、50に指定することですぐに変わるロールオーバーを実現できます。
onComplete
アニメーションが完了した際に呼び出されるコールバック関数です。
マウスオーバー・マウスアウト時の両方で呼び出されます。
onHover
マウスオーバー時に呼び出されるコールバック関数を指定します。
onOut
マウスアウト時に呼び出されるコールバック関数を指定します。
