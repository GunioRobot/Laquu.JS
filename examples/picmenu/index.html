Laquu.JS ピックメニュー

シンプルなピックメニュー

Code

jQuery(function(){
    laquu(".pickmenu").picmenu();
});

オプション付きピックメニュー

Code

jQuery(function(){
    laquu(".pickmenu").picmenu({
        showSize: 180,
        hideSize: 40,
        speed: 600,
        easing: "easeInOutExpo",
        isVertical: true,
        auto: true
    });
});
