シンプルなツールチップ

Code

jQuery(function(){
    laquu(".tooltip").tooltip();
});

ここにマウスを載せてください。

オプション付きツールチップ

Code

jQuery(function(){
    laquu(".tooltip").tooltip({
        distX: -20,
        distY: -30,
        onShow: $l.empty,
        onHide: $l.empty,
        onMove: $l.empty
    });
});

ここにマウスを載せてください。

指定可能オプション

distX
X軸の表示位置を調整する値です
distY
Y軸の表示位置を調整する値です
onShow
ツールチップが表示された際に呼び出されるコールバック関数です
onHide
ツールチップが消えた際に呼び出されるコールバック関数です
onMove
ツールチップが移動するたびに呼び出されるコールバック関数です
