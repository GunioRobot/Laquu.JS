Laquu.JS ティッカープラグイン

シンプルなティッカー

Code

jQuery(function(){
    laquu(".ticker").ticker();
});
  • アネモネは交響詩篇エウレカセブンに登場する敵キャラクタです。
  • エウレカと同じ人型コーラリアンで、ホランド・ノヴァクの兄、デューイ・ノヴァクによって作られた人型コーラリアンです。
  • アネモネはエウレカたちと戦いを繰り返していく内に、自分のドミニクへ対する感情が現れていきます。

オプション付きティッカー

Code

jQuery(function(){
    laquu(".ticker").ticker();
});
アネモネは交響詩篇エウレカセブンに登場する敵キャラクタです。
エウレカと同じ人型コーラリアンで、ホランド・ノヴァクの兄、デューイ・ノヴァクによって作られた人型コーラリアンです。
アネモネはエウレカたちと戦いを繰り返していく内に、自分のドミニクへ対する感情が現れていきます。

利用可能オプション

speed
ティッカーのアニメーション速度を指定します
duration
アニメーションを開始するまでの時間を指定します
onShow
ティッカーが表示（アニメーション完了）後に呼び出されるコールバック関数です

※ 画像サイズ及び、各要素にはCSSなどで高さを指定してください。
