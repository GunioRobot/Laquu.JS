バブルポッププラグイン｜Laquu.JS

シンプルな動作例

Code

$(function(){
    laquu("#bubble1").bubblepopup();
});

バブルポップアップを起動

これがバブルポップアップ

邪魔なコンテンツを整理することが出来ます。

オプション付き動作例

Code

$(function(){
    laquu("#bubble2").bubblepopup({
        easing: "easeInOutBack",
        distance: 50,
        distanceStep: 10,
        hideDelay: 600,
        popupClass: ".popup",
        triggerClass: ".trigger",
        onShow: function(popup, trigger) {
            popup.animate({ fontSize: "2.4em" });
        },
        onHide: function(popup, trigger) {
            popup.animate({ fontSize: "100%" });
        }
    });
});

バブルポップアップを起動

これがバブルポップアップ

邪魔なコンテンツを整理することが出来ます。
