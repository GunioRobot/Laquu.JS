コナミプラグイン｜Laquu.JS

シンプルな動作例

Code

jQuery(function(){
    laquu(document).konami();
});

上上下下左右左右BAとキーボードで入力してください。

オプション付き動作例

Code

jQuery(function(){
    laquu(document).konami({
        cmd: "38,38,40,40,66,65",
        callback: function(el, ev, stack) {
            alert("push commands " + stack.toString() + " pushed !");
        }
    });
});

上上下下BAとキーボードで入力してください。
