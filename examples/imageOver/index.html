イメージオーバープラグイン｜Laquu.JS

シンプルな動作例

Code

jQuery(function(){
    laquu("#imageOver1 img").imageOver();
});

オプション付き動作例

Code

jQuery(function(){
    laquu("#imageOver2 img").imageOver({
        suffix: "_over",
        onHover: function() {},
        onOut: function() {}
    });
});
